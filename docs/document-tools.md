# Free Document Tools

FreeProTool includes document tools for common browser-based file workflows.

Examples may include tools for DOCX, ODT, TXT, Markdown, EPUB, and related document formats.

Live document tools:
https://freeprotool.com/document-tools

## Bug reports

If a document tool fails, please report the tool name, browser, operating system, document format, approximate file size, and what happened.

Do not upload private documents to GitHub issues.
