# Free PDF Tools

FreeProTool includes PDF tools for common browser-based document workflows.

Examples include:

- Merge PDF
- Split PDF
- Compress PDF
- Rotate PDF
- Remove PDF metadata
- Remove images from PDF
- Convert PDF pages where supported

Live PDF tools:
https://freeprotool.com/pdf-tools

## Bug reports

If a PDF tool fails, please report:

- Tool name
- Browser
- Operating system
- Approximate file size
- Whether the PDF is scanned, encrypted, image-heavy, or form-based

Do not upload private PDFs to GitHub issues.
