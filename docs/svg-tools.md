# Free SVG Tools

FreeProTool includes SVG tools for viewing, editing, optimizing, minifying, and cleaning SVG files.

Examples include:

- SVG viewer
- SVG editor
- SVG optimizer
- SVG minifier
- Remove SVG metadata
- Reduce SVG file size

Live SVG tools:
https://freeprotool.com/svg-tools

## Bug reports

If an SVG tool fails, please report the tool name, browser, operating system, approximate file size, and whether the SVG includes scripts, embedded images, masks, filters, or complex paths.

Do not upload private SVG files to GitHub issues.
