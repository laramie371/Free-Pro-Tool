# Privacy

FreeProTool is designed around simple, browser-based tools.

Where possible, tools process files locally in your browser. This means the file can be handled on your device without needing to upload it to a server.

Some future tools or AI-powered tools may require server-side processing. When that is the case, the tool page should clearly explain it.

## Important safety note

Do not upload private files, passwords, tax documents, medical records, legal records, financial records, or personal identity documents to public GitHub issues.

If a bug involves a private file, describe the problem without attaching the file.
