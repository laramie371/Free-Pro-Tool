# Free Data Tools

FreeProTool includes data tools for developer and dataset workflows.

Examples include tools for:

- CSV
- JSON
- XML
- YAML
- TSV
- SQL
- LOG
- INI
- CONF

Live data tools:
https://freeprotool.com/data-tools

## Bug reports

If a data tool fails, please report the tool name, browser, operating system, file type, approximate file size, and whether the file has unusual encoding or delimiters.

Do not upload private datasets, secrets, API keys, or customer data to GitHub issues.
