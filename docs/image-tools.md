# Free Image Tools

FreeProTool includes browser-based image tools for common image cleanup, conversion, optimization, and editing workflows.

Examples include:

- Resize images
- Compress images
- Convert image formats
- Remove image metadata where supported
- Crop, rotate, flip, or mirror images where supported

Live image tools:
https://freeprotool.com/image-tools

## Bug reports

If an image tool fails, please report the tool name, browser, operating system, image format, approximate file size, and what happened.

Do not upload private or sensitive images to GitHub issues.
