# Contributing

Thanks for helping improve FreeProTool.

This repository contains public documentation, examples, metadata, and safe utility code. The production FreeProTool app source code is private.

## Good contributions

- Bug reports
- Tool requests
- Documentation fixes
- Example improvements
- File type notes
- Browser compatibility reports

## Please do not upload private files

Do not attach private, sensitive, financial, medical, legal, or personal files to GitHub issues.

If a file causes a bug, describe the file type, size, source app, and behavior instead.

## Reporting bugs

Use the Bug Report or File Processing Problem issue form.

## Requesting tools

Use the Tool Request issue form.
