# FreeProTool Roadmap

This is the public roadmap for FreeProTool.

## Current focus

- PDF tools
- Image tools
- SVG tools
- Document tools
- Data tools
- Browser-based local processing
- Clear privacy information
- Better file examples and datasets

## Planned improvements

- More file conversion tools
- Better SVG optimization tools
- More PDF cleanup tools
- Improved mobile layouts
- More sample files and datasets
- Optional AI-assisted tools with clear privacy notices

## Tool request process

Use GitHub Issues to request new tools:
