# FreeProTool

FreeProTool is a free browser-based toolkit for working with PDFs, images, SVGs, documents, data files, and developer formats.

This public repository contains documentation, tool metadata, safe example code, issue templates, and the public roadmap for FreeProTool.

The production FreeProTool website source code is not included in this repository.

## Live site

https://freeprotool.com

## What is included here?

- Public tool catalog
- File type documentation
- Browser utility examples
- Safe standalone tool demos
- Public roadmap
- Bug report and tool request system
- Privacy and local-processing documentation

## What is not included?

- Production website source code
- Internal SEO system
- Admin tools
- Full UI components
- Private tool engine
- Analytics or monetization code

## Tool categories

- PDF tools
- Image tools
- SVG tools
- Document tools
- Data tools
- Developer tools

## Report a bug

Use GitHub Issues to report bugs, broken tools, file processing problems, or incorrect page information.

Please do not upload private, sensitive, financial, medical, legal, or personal files to GitHub issues.

## Request a tool

You can request new tools through the Tool Request issue form.

## Privacy

Many FreeProTool tools are designed to run locally in your browser where possible. See `docs/privacy.md` and `docs/local-browser-processing.md` for more information.
